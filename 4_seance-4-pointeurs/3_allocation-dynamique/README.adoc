= Exercice : Manipulation dynamique de mémoire

À partir des définitions présentes dans main.h :

. Retournez un tableau de la taille demandée par l'utilisateur. Toutes ses valeurs sont initialisée à 0 :
`int * createTab(int size);`

. Écrire une fonction qui demande à l'utilisateur le nombre de case d'un tableau de char à trois dimensions, lui demande de le remplir puis l'affiche :
`void userTab3D();`
